# Minerva — Aesthetic Archive

Minerva is a personal aesthetic curation web app that allows users to collect, organize, and analyze visual inspiration.

## ✨ Features
- Save images (local upload)
- Save links with preview (microlink API)
- Organize into boards
- Tagging & filtering
- Insight panel (usage analytics)
- Local storage persistence

## 🛠 Tech Stack
- HTML / CSS / JavaScript
- localStorage (no backend)
- Microlink API (link preview)

## ⚠️ Notes
Data is stored locally in browser storage. Clearing cache will remove saved items.
